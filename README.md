# Gerador de Prompt para Agente de Ambientes Realistas

Produto HTML offline para gerar prompts detalhados de design de interiores a partir de uma foto real do ambiente.

## Como usar

1. Abra o `index.html` no navegador.
2. Escolha o ambiente, estilo, modelo de proposta, paleta, materiais e prioridades.
3. Anexe ou use uma foto real do ambiente como referencia.
4. Copie o prompt gerado e cole no GPT junto com a imagem.

## O que o gerador entrega

- Prompt para usar com foto real do ambiente.
- Instrucoes para criar um agente GPT especializado.
- Modelos de proposta: economica, intermediaria, premium, antes/depois, checklist e prompts de imagem.
- Briefing com limitacoes de obra, materiais, ergonomia, custos e prioridades.

Funciona 100% offline.
